# 更新履歴

## v1.1 (2026-02-06)

UIに関する改善を加えました。

- 詳細メニュー: 「すぐ下に追加」する機能を追加しました。また、画面外に切れにくい表示に改善し、外側クリック / Esc で閉じるようにしました。
- 追加バー: 追加バーが追随するようになりました。スクロール中も読みやすい背景になりました。
- ラベル: SONG / NOTE / ENCORE の色分けをおこないました。
- アンコール: 操作できない状態では無効（disabled）に見えるようにしました。
- セットリスト編集: 画面幅に応じて横幅が自動調整されるレイアウトに改善しました。
- タイトルサジェスト: 実験的機能として追加しました（稀に正常に表示されないことがあります）。
- テーマ設定: テーマ設定でライトモードとダークモードを切り替えられるようにしました。
- UI微修正: 全体的に使いやすいように調整しました。
